                           2. РОЛИ В КАНАЛЕ (ChannelRole)
═══════════════════════════════════════════════════════════════════════════════

┌─────────────────┬─────────────────────────────────────────────────────────────┐
│ РОЛЬ В КАНАЛЕ   │ ПРАВА ВНУТРИ КАНАЛА                                          │
├─────────────────┼─────────────────────────────────────────────────────────────┤
│ Owner           │ • Полные права в канале                                      │
│ (Владелец)      │ • Может удалять канал                                        │
│                 │ • Может менять роли участников                               │
│                 │ • Может удалять ЛЮБЫЕ сообщения                              │
│                 │ • Может приглашать/удалять участников                        │
│                 │ • НЕ может покинуть канал без передачи прав                  │
├─────────────────┼─────────────────────────────────────────────────────────────┤
│ Admin           │ • Может удалять ЛЮБЫЕ сообщения                              │
│ (Администратор) │ • Может приглашать/удалять участников                        │
│                 │ • Может менять роли (кроме Owner)                            │
│                 │ • Может покинуть канал                                       │
├─────────────────┼─────────────────────────────────────────────────────────────┤
│ Moderator       │ • Может удалять ЛЮБЫЕ сообщения                              │
│ (Модератор)     │ • НЕ может приглашать/удалять участников                     │
│                 │ • НЕ может менять роли                                       │
│                 │ • Может покинуть канал                                       │
├─────────────────┼─────────────────────────────────────────────────────────────┤
│ Member          │ • Может отправлять/редактировать СВОИ сообщения              │
│ (Участник)      │ • Может удалять ТОЛЬКО свои сообщения                        │
│                 │ • Может покинуть канал                                       │
└─────────────────┴─────────────────────────────────────────────────────────────┘

                        4. ОСНОВНЫЕ БИЗНЕС-ПРОЦЕССЫ
═══════════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.1. СОЗДАНИЕ КАНАЛА                                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1. Пользователь отправляет запрос с:                                        │
│    - Названием канала (обязательно, уникально)                              │
│    - Описанием (опционально)                                                │
│    - Списком приглашаемых пользователей                                     │
│    - Сообщением приглашения (опционально)                                   │
│                                                                             │
│ 2. Проверки:                                                                │
│    ✓ Название канала уникально                                              │
│    ✓ Приглашающий имеет право приглашать выбранных пользователей             │
│    ✓ Пользователь не приглашает сам себя                                    │
│    ✓ Нет активного приглашения для этого пользователя                        │
│    ✓ Пользователь еще не участник канала                                    │
│                                                                             │
│ 3. Результат:                                                               │
│    - Канал создан                                                           │
│    - Создатель становится Owner'ом                                          │
│    - Приглашенным отправляются уведомления                                  │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.2. ПРИГЛАШЕНИЕ ПОЛЬЗОВАТЕЛЕЙ В СУЩЕСТВУЮЩИЙ КАНАЛ                         │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1. Требуемые права: Owner или Admin                                         │
│                                                                             │
│ 2. Проверки:                                                                │
│    ✓ Приглашающий в канале Owner или Admin                                  │
│    ✓ Приглашаемый не участник канала                                        │
│    ✓ Нет активного приглашения                                              │
│    ✓ Приглашающий может приглашать эту роль                                  │
│                                                                             │
│ 3. Результат:                                                               │
│    - Создано приглашение со статусом Pending                                │
│    - Приглашенному отправлено уведомление                                   │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.3. ПРИНЯТИЕ ПРИГЛАШЕНИЯ                                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1. Проверки:                                                                │
│    ✓ Приглашение существует                                                 │
│    ✓ Приглашение адресовано текущему пользователю                           │
│    ✓ Статус = Pending                                                       │
│    ✓ Приглашение не истекло (до 7 дней)                                     │
│    ✓ Канал существует и активен                                             │
│    ✓ Пользователь еще не участник                                           │
│                                                                             │
│ 2. Результат:                                                               │
│    - Статус приглашения → Accepted                                          │
│    - Пользователь добавлен как Member                                       │
│    - Отправителю отправлено уведомление                                     │
│    - Участники канала уведомлены через Hub                                  │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.4. ОТКЛОНЕНИЕ/ОТМЕНА ПРИГЛАШЕНИЯ                                         │
├─────────────────────────────────────────────────────────────────────────────┤
│ Отклонение (получателем):                                                   │
│    - Статус → Declined                                                      │
│    - Отправителю уведомление                                                │
│                                                                             │
│ Отмена (отправителем/Owner/Admin):                                          │
│    - Приглашение удаляется из БД                                            │
│    - Получателю уведомление                                                 │
│    - Требуются права: отправитель ИЛИ Owner ИЛИ Admin                       │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.5. ПОКИДАНИЕ КАНАЛА                                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1. Проверки:                                                                │
│    ✓ Пользователь НЕ Owner (Owner должен передать права)                    │
│    ✓ Пользователь является участником                                       │
│                                                                             │
│ 2. Результат:                                                               │
│    - Участник удален из канала                                              │
│    - Участники уведомлены через Hub                                         │
│    - Если участников = 0 → канал удаляется                                  │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.6. УДАЛЕНИЕ УЧАСТНИКА ИЗ КАНАЛА                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1. Требуемые права: Owner или Admin                                         │
│                                                                             │
│ 2. Проверки:                                                                │
│    ✓ Удаляющий имеет права Owner/Admin                                      │
│    ✓ Удаляемый НЕ Owner                                                     │
│    ✓ Удаляемый является участником                                          │
│                                                                             │
│ 3. Результат:                                                               │
│    - Участник удален                                                        │
│    - Если участников = 0 → канал удаляется                                  │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.7. ИЗМЕНЕНИЕ РОЛИ УЧАСТНИКА                                              │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1. Требуемые права: Owner или Admin                                         │
│                                                                             │
│ 2. Проверки:                                                                │
│    ✓ Изменяющий имеет права Owner/Admin                                     │
│    ✓ Изменяемый НЕ Owner (нельзя изменить роль Owner)                       │
│    ✓ Изменяемый является участником                                         │
│                                                                             │
│ 3. Результат:                                                               │
│    - Роль участника изменена                                                │
│    - Доступные роли: Member, Moderator, Admin                               │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.8. ПРОСМОТР ДОСТУПНЫХ ПОЛЬЗОВАТЕЛЕЙ ДЛЯ ПРИГЛАШЕНИЯ                       │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1. Автоматическая фильтрация по роли текущего пользователя:                 │
│    - Student/Parent → видят ТОЛЬКО Student и Parent                         │
│    - Teacher/Admin → видят ВСЕХ                                             │
│                                                                             │
│ 2. Автоматически исключаются:                                               │
│    ✓ Текущие участники канала                                               │
│    ✓ Пользователи с активными приглашениями                                 │
│                                                                             │
│ 3. Поддерживается:                                                          │
│    - Поиск по имени/фамилии/email                                           │
│    - Фильтрация по роли (в рамках доступных)                                │
│    - Фильтрация по группе                                                   │
│    - Пагинация и сортировка                                                 │
└─────────────────────────────────────────────────────────────────────────────┘

                             8. ВАЖНЫЕ ПРАВИЛА
═══════════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────────┐
│ 1. СТУДЕНТЫ И РОДИТЕЛИ НЕ МОГУТ:                                            │
│    • Видеть учителей и администраторов в списке доступных пользователей     │
│    • Приглашать учителей и администраторов                                  │
│    • Фильтровать по роли Teacher/Administrator                              │
├─────────────────────────────────────────────────────────────────────────────┤
│ 2. ВЛАДЕЛЕЦ КАНАЛА (OWNER):                                                 │
│    • Не может покинуть канал (должен передать права)                        │
│    • Не может быть удален из канала                                         │
│    • Его роль нельзя изменить                                               │
├─────────────────────────────────────────────────────────────────────────────┤
│ 3. СООБЩЕНИЯ:                                                               │
│    • При удалении сообщения удаляется и его изображение                     │
│    • Редактировать можно только свои сообщения                              │
│    • Удалять чужие могут: Owner, Admin, Moderator                           │
│    • При полном удалении канала удаляются все сообщения и файлы             │
├─────────────────────────────────────────────────────────────────────────────┤
│ 4. ПУСТОЙ КАНАЛ:                                                            │
│    • Если в канале не осталось участников - канал автоматически удаляется   │
├─────────────────────────────────────────────────────────────────────────────┤
│ 5. ПРИГЛАШЕНИЯ:                                                             │
│    • Автоматически истекают через 7 дней                                    │
│    • Нельзя отправить приглашение уже приглашенному пользователю            │
│    • Нельзя пригласить уже состоящего в канале пользователя                 │
└─────────────────────────────────────────────────────────────────────────────┘

                           10. ПОЛЕЗНЫЕ ЗАПОМНИТЬ
═══════════════════════════════════════════════════════════════════════════════

• GetCurrentUserId() - получить ID текущего пользователя из Claims
• _currentUserService.UserId - альтернативный способ
• ChannelRole (в канале) ≠ UserRole (системная роль)
• IsRead отмечается ТОЛЬКО для получателя (не для отправителя)
• При создании канала создатель автоматически становится Owner и Member
• FileService.SaveImageAsync() возвращает относительный путь
• FileService.DeleteImageAsync() удаляет файл по URL
• Все исключения на русском языке
• Пагинация через FilterEntity (Page, PageSize, SortBy, SortOrder, SearchTerm)
